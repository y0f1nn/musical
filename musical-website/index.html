Installation Guide

Since this is a Github preview, you'll need to load it manually. It takes less than a minute.

01

Download & Unzip

Download the musical.zip file above and extract it to a folder on your computer. Remember where you put it!

02

Open Extensions

Open Chrome and navigate to chrome://extensions in your address bar.

03

Developer Mode

Toggle the Developer mode switch in the top right corner of the page.

04

Load Unpacked

Click the Load unpacked button that appears. Select the folder you extracted in Step 1.
